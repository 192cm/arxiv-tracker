# my_save

이번 세션에서 일어난 작업을 검토하고 CLAUDE.md에 분산 저장해줘.

## 수행 절차

### 1. 세션 작업 검토
이번 세션에서 실제로 일어난 일을 추려낸다:
- 새로 작성/수정한 파일
- 실행한 명령과 결과
- 내린 결정과 그 이유
- 새로 발견한 사실, 통찰
- 미완 작업과 다음에 이어갈 항목

### 2. CLAUDE.md에 반영

**CLAUDE.md에는 프로젝트 규칙으로 승격할 것만 추가:**
- 앞으로 **반복해서** 적용될 규칙 ("이 프로젝트는 X 방식을 쓴다")
- 새로 확립된 컨벤션, 금기사항
- 빌드/테스트 명령 추가, 변경
- 아키텍처적 결정

**추가하지 말 것:**
- 일회성 작업 내역
- 오늘만의 맥락
- 이미 존재하는 내용과 중복

기존 섹션 구조를 존중하고, 적합한 헤더 아래에 추가한다. 추가할 게 없으면 수정하지 말고 그 이유를 사용자에게 알린다.

### 3. 작성 후 사용자에게 한 줄로 요약 보고

"CLAUDE.md에 다음을 추가했습니다: ..." 또는 "추가할 영구 규칙이 없어 수정하지 않았습니다."
